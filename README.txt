Mini chat PHP
=============


Petit chat simple réalisé en PHP par Axel Puig pour l'activité du cours sur PHP et MySQL de Mathieu Nebra, avec l'utilisation d'une base de données MySQL. En cas d'erreur, tout est expliqué.


Important pour être sur que tout fonctionne !
---------------------------------------------

- Soyez sûr d'avoir bien mis le dossier dans le répertoire de WAMPP (ou équivalent)
- Pensez à importer la base de données
